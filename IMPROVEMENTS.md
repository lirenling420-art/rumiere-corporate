- **CTAボタンの改善**
  - ホバー時にスケール＆シャドウ効果
  - 高級感のある見た目

- **スクロールインジケーター**
  - ユーザーにスクロール促進
  - 上品なアニメーション

#### ValuesSection.tsx
- 企業の3つの価値観（信頼、品質、成長）を表示

#### BusinessCategoriesSection.tsx
- 6つの事業カテゴリをグリッド表示

- カード型デザインで整理感を強化

- アイコンが回転＆スケール
- 「詳しく見る」テキストがスライド

#### BeautyHighlightSection.tsx
- 美容・サロン事業を特別にハイライト
- 左右レイアウトで視覚的な変化
- ホバー時にシャドウ効果
- アニメーション付きのビジュアルエリア

#### CTASection.tsx
- 最終的なお問い合わせ導線
- チャコール背景に金色のボタン
- 背景にゆっくり動くゴールドの光
- 高級感のある見た目

### 3. デザイン改善

**配色**
- 白、アイボリー、グレージュ、チャコール、控えめなゴールド
- 会社公式サイトとしての信頼感を維持
- 美容ブランドのような上品さを加える

**タイポグラフィ**
- Noto Serif JP（見出し）
- Noto Sans JP（本文）
- フォントウェイトで強弱をつける
- 読みやすさを最優先

**余白**
- セクション間に広い余白（py-32）
- 上品で落ち着いた印象
- 情報の整理感を強化

**アニメーション**
- すべてのアニメーションが滑らか
- 安っぽい動きは排除
- 会社公式サイトらしい上品さを維持
- Framer Motion の `whileHover`, `whileInView`, `animate` を活用

### 4. インタラクション

- **ホバー効果**
  - ボタン：スケール＆シャドウ
  - カード：上昇＆シャドウ
  - テキスト：スライド

- **スクロール時の演出**
  - `whileInView` で要素が見えた時にアニメーション
  - `viewport={{ once: true }}` で1回のみ実行

- **ページロード時の演出**
  - ステップ遅延でコンテンツが順番に表示
  - 統一感のある動き

### 5. レスポンシブ対応

- すべてのコンポーネントがモバイル対応
- タブレット、デスクトップでも最適表示
- グリッドレイアウトで自動調整
